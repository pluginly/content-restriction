## Installation and Usages

1. Clone Github Repository

   ```sh
   git clone https:://github.com/pluginly/content-restriction.git
   ```

2. Install composer dependencies
   ```sh
   composer install
   ```

3. Install npm dependencies (Node v18.x)
   ```sh
   npm install
   ```

4. Compile Scripts
   ```sh
   npm start
   ```

4. Build plugin
   ```sh
   npm run build
   ```
